== Description ==

Vous souhaitez mettre un peu d'originalité dans la présentation des visuels/illustrations que vous affichez sur votre site web ? WIP est là pour vous !

Wordpress Interactive Pictures vous permet de dynamiser votre site web en annotant vos images à l'aide de drag & drop. Vous pourrez ajouter des annotations sur vos images et y insérer un titre, une image, une icône et une couleur de pointeur ainsi qu'un descriptif complet et personnalisé.
Ajoutez à cela l'intégration dans vos contenus avec le système de shortcode de Wordpress (avec l'intégration d'un bouton personnalisé dans l'éditeur de texte) pour donner un peu d'originalité à vos contenus !

== Installation ==

Comment installer le plugin - How to install.

1. Upload the `wp-interactive-pictures` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the WP Interactive Pictures tab and add your pictures
4. Enjoy !
