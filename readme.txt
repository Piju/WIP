== Description ==

You always wanted to add orignal pictures with annotation in your Wordpress website content ? Wordpress Interactive Pictures (WIP) is here to help you do this.

With Wordpress Interactive Pictures you'll be able to show original annoted pictures with FontAwesome/Glyphicons icons with drag & drop on your pictures. After that, you can add annotation with title, description, thumbnail and custom colored cursor.
Finally, you can insert it with a simple shortcode which allows you to load your picture with different templates to show entire world original contents !

== Installation ==

Comment installer le plugin - How to install.

1. Upload the `wp-interactive-pictures` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the WP Interactive Pictures tab and add your pictures
4. Enjoy !
